AMR2FRED REST Service

A paper describing AMR2FRED has been accepted for publication to the ESWC Conference where it won the best demo award. A standalone application can be extracted from here.

Select output type option to get proper help.

Base URI:

Parameter Explanation
&?amr=(x) (x) is the AMR encoded with UTF-8
&?format=(f) (f)select the output format - it is one from ( RDF_XML | RDF_XML_ABBREV | N_TRIPLES | TURTLE | DIGRAPH | png )
&?rid_err=ON if present it tells to amr2fred to remove nodes with translation errors
&?objAsRes if present it tells to amr2fred to insert third parts of rdf statements as resources instead of as strings
